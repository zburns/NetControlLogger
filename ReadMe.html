Net Control Logger - Documentation

Features

  • Customizable Startup Logo - Use your club's logo, Skywarn Logo, ARES Logo, etc
    • Just create a 440 pixel wide by 240 pixel high "Logo.jpg" in the same directory as the program files
    • Image will auto-scale if needed
  • System Designed for Microsoft Windows with no "installation or installer needed"
    • Runs on Windows 98SE, Windows ME, Windows 2000, Windows XP, Windows Vista, Windows 7
    • Only requirement is Microsoft .Net Framework 2.0 or later (with comes with all the above OS)
    • All Files needed to run are contained in distribution ZIP file.
      • "When all else fails" approach.  Just a text editor needed (notepad, etc)
  • Will Log events
    • Skywarn Net Control Changes, Net Condition Changes, Station Check In/Out Times, etc
  • Will use Tactical Calls if needed
  • If Internet connection, will pull from FCC/QRZ, etc. databases.
  • Ability to have Net Control scripts and launch them from withing the Net Control software
  • Quickly use Enter/Tabs to Check Stations In
    • Hands on approach for when the check-ins really fly
